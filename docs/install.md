# Install

## Linux

```sh
chmod +x cherry-linux-x86_64
./cherry-linux-x86_64 bootstrap --yes
```

Cherry installs to:

```text
~/.cherry/bin/cherry
```

## Windows PowerShell

```powershell
.\cherry-windows-x86_64.exe bootstrap --yes
```

Cherry installs to:

```text
%USERPROFILE%\.cherry\bin\cherry.exe
```

Restart your terminal after bootstrap so the updated PATH is visible to new
shells.
